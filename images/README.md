# Extension Icon

Place a 128x128 PNG icon named `icon.png` in this directory.

The icon should:
- Be 128x128 pixels
- Have a transparent background
- Represent debugging/MCP concepts
- Follow VS Code icon guidelines

For now, you can create a simple icon or use a placeholder.

## Creating an Icon

You can use tools like:
- Figma
- Adobe Illustrator
- Inkscape (free)
- Online icon generators

## Icon Guidelines

- Simple and recognizable
- Works well at small sizes
- Matches VS Code's visual style
- Uses appropriate colors for debugging theme
